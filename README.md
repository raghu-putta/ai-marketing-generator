## 📌 Project Overview
This project is a FastAPI-based application that generates marketing content based on user inputs such as product, audience, and tone.

---

## ✨ Features
- Generates Facebook Ads
- Creates Email Campaigns
- Writes Blog Introductions
- REST API with FastAPI
- Interactive Swagger UI for testing
- Dockerized and deployed on AWS EC2

---

## 🛠 Tech Stack
- Python
- FastAPI
- Docker
- AWS EC2
- REST API
