# AI Marketing Content Generator 
## Live Demo
http://34.224.216.204/docs

This project is a FastAPI-based application that generates marketing content based on user inputs.

## Features
- Generates Facebook Ads
- Creates Email Campaigns
- Writes Blog Introductions
- Accepts product, audience, and tone as input

## Tech Stack
- Python
- FastAPI
- REST API
- Prompt-based content generation

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the server:
   uvicorn main:app --reload

3. Open in browser:
   http://127.0.0.1:8000/docs

## Example Input
- Product: Fitness App
- Audience: Young Professionals
- Tone: Energetic
